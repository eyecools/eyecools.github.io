A FEW STEPS A FEW DEEP DEGREES

每一个不曾起舞的日子都是对生命的辜负。

第一篇博客

城市年轻人饱受日益拥堵的公共交通困扰，电动车，不但节约时间、精力，2022冬奥举办，更能为环境保护做贡献，减霾减排，换成具体数据就是4200万吨碳排量，与城市种植70亿棵大树等效。

我们是否可以想象：如果每个人城市上下班通勤都将汽车改用两轮电动车，那将释放多少个5座汽车的空闲座位？！

2016年4月21日牛电科技在798艺术区 D-Park 举办了主题为“科技玩转城市 Turn the city on”的小牛 M1 发布会。

〓 第一章： 外观、细节 〓

▲车载 USB—20Ah“移动电源”

骑着小牛出门不怕手机没电了，小牛车载 USB 可随时为手机和电子产品充电。强迫症朋友们不用看一下导航就关 GPS 和网络图省电了，驾驶者可以边导航边充电。

USB 口右侧挂钩可以挂早餐，着急怕堵车的时尚上班族们，可别不吃早餐。

车载 USB 接口位于电门钥匙和挂钩旁，位置方便。贴心的 M1 在钥匙口上方加装了微光灯，漆黑夜里再也不用为找不准钥匙孔方向抓狂了，赞。

USB 口的参数是 5V，最大 1A。实际测试空载电压为 5.25V，给手机充电时电流 0.92A 电压 4.97V。

USB 采用 DC40-100V 独立模块，因此在牵引前进电机大电流模式下，USB 输出电压 4.95V，电流 0.90A，输出电压电流平稳，可放心给电子产品充电。

小牛 USB 供电就像如今购买较为人性化的汽车一样，有它，充电很方便。有了这个“小牛移动大电源”在骑行中可以随时查看导航、听歌，不用再为手机没电紧张了。

限速 20km/h 或 45km/h 也是由小牛盒子通过逻辑控制(并非传统电动车通过拔限速线缆实现），出厂限速 20km/h，是因为根据国家标准化管理委员会颁布的《电动自行车通用技术条件》，电动车的最高时速不大于20公里每小时。M1 解除限速方法很有趣，需要配合电门、刹车把手、启动按钮等操作，就像当年玩魂斗罗的上上下下 AABB 一样，详情可上网搜索，或者私信 Vitas。（新浪微博：胡小哥 Vitas）

需要吐槽的是：缩小后的紧凑车身没有一个储物空间，座筒空间也基本被电池占据（座筒剩余储物空间可以放入一个1米长的链条锁），就连 380mL 小瓶水都没办法放入。无耐，只能将水瓶塞入背包或者口袋

建议厂家之后的量产车能像 N1 那样，保留前部小储物盒，或者预留加装位置和接口。

全新设计的自动感光 LED 一体式灯组赶上豪华轿车的配置，采用亲和性的圆形造型，同时兼顾了日间行车灯与远近光照明的功能融合设计。由一条极具高科技感的 LED 灯条与高度可视化刹车灯构成的尾灯组完美镶嵌于车体，提供全角度照明。环形天使眼与定制滚边圆角透镜，配合 LED 灯珠在为您提供安全性的同时带来超广角的照明区域，三颗球型光组晚上骑出去照明效果确实不错。

感光自动大灯：将大灯放置在 A 档，手把显示器左右两侧有两个红外感光传感器，当环境光照亮度到达传感器设定的门槛值约1秒钟，大灯自动关闭。反之，环境亮度到达门槛值以下，大灯会马上开启，桥洞、地下停车场、隧道等环境很实用。真是酷炫环保又安全。

▲车辆防水

为了适应更多的驾驶环境，M1 的主要电子元件都具备超高防护等级。电池包、电机控制器、电源转换器、大灯、仪表、防盗器、中控及可遇水线缆的防水防尘系数均高达 IP65*。200mm 的涉水深度，保障雨天安全骑行。小牛 M1 的锂电池放置坐筒里，油门调节器放置在了车头下的控制箱内，外边用拉线连接。这样可以防止电路部分手把出淋雨短路而无法骑行。

▲后挡泥板需要优化

为了下雨天更好防止后轮甩污水，后挡泥板被加长且多伸出了个檐。个人觉得既不美观也不实用。

厂家可否考虑将挡泥板设置成以轴孔固定点为圆心可旋转伸缩双层设计，晴天，将后半截挡泥板收进去，阴雨天需要使用时候轻轻按卡口就会伸长。（弱弱的问一句：Vitas 是不是可以申请挡泥板专利？）

调整后的挡泥板倒车调头时也不会经常碰到挡泥板，最大的优势是：轴距不变的情况下整车缩短 9cm 的长度对于将要把便捷的 M1 塞进升降梯的用户来说，还是非常有用的。

工程试装车的前仪表盘亮度过高，白天没有问题，晚上行驶在光线较暗环境下，比较强的仪表灯会有点刺眼，希望厂家尽快优化。

‍‍〓 第二章：乘客安全、车辆安全 〓

▲乘客安全

优良的制动（刹车）系统，是行车安全的重要保障，车，让他走起来容易，让他能按照驾驶者的意图丝毫不差停止在指定区域，是一个重要课题。

就如同 Vitas 从事的轨道交通车辆控制工作，高铁和地铁等轨道车辆最高跑多少时速不是由牵引力决定，而是由制动——在规定距离和区间内能够停下来而决定车辆最高运行速度，当然，轨道和轮对踏面之间的粘着系数远远低于轮胎和柏油路面。

因此，一个车辆的制动性能，很大程度决定了乘客的安全，下面就和 Vitas 一起来看看小牛的制动（刹车）工况。

前制动方式：180mm 刹车盘搭配单向双活塞卡钳

后制动方式：鼓刹

制动，俗称“刹车”。现在市面上较为常见的制动系统有两大类，分别是碟式刹车和鼓式刹车。

碟式刹车的主要优点是在高速刹车时能迅速制动，散热效果优于鼓式刹车，制动效能的恒定性好，使用金属块(碟)而不用鼓轮，在刹车碟的两边都有一平坦的刹车蹄，当刹车总泵来的油压压送到分缸，使刹车蹄向刹车碟夹住，以达到刹紧的效果，如今已普遍用于汽车前轮，有的高级车装置四轮碟式刹车，其优点是作用灵敏，散热良好，不必调整刹车间隙，保养容易。

鼓式刹车有良好的自刹作用，由于刹车来令片外张，车轮旋转连带着外张的刹车鼓扭曲一个角度(当然不会大到让你很容易看得出来)刹车来令片外张力(刹车制动力)越大，则情形就越明显。而且鼓刹制造技术层次稍低，也是最先用于刹车系统，因此制造成本要比碟式刹稍低。

现如今的中高端汽车都采用碟(盘）式制动，高铁车辆也不例外。

小牛 M1 经过精心选择最终确认使用的高锰钢刹车片、博世刹车油及金属编织输油管，大大提升了刹车系统的使用寿命和稳定性。液压传动使得刹车更加灵敏。

小牛 M1 试装车刹车盘：OOX-3mm ，可以选择盘刹孔径 6.6MM 以下的碟盘锁（很多心热的朋友，车还没发货，锁都买好了）。

可能是为了减轻整车重量，M1 的后置刹车使用鼓刹，实际使用比较灵敏，稍微加力后轮便会抱死。对于 57kg 车身且只能乘坐一位乘客的 M1，制动反应足够。

两轮车一般直线制动，由于重心向前，前轮粘着力带来的摩擦力占主要减速效率（尤其在高速行驶的时候）。行驶中的前刹会把车子的惯性转变为向下的力，这时，前轮会获得比后轮更大的摩擦力。从而更快的停下来。从安全制动角度讲，高速行驶制动工况后刹都会抱死，产生侧滑，而前轮，只要车辆直立制动用力不是特别离谱，一般不会侧滑，所以前刹更安全，制动时候按力分配应该是前6后4。Vitas 也提醒朋友们，一定不要在转弯的时候用力制动，以免甩尾、侧滑、摔倒。

小牛官方是这样说的:

制动灵敏，拥有能量回收功能的刹车系统，EBS 电子刹车+动能回收系统，制动性能突出。

紧急制动时轮胎状态，由于惯性向前，后轮已经开始滑行。(制动视频测试见文章尾）。

多次测试轮盘和轮胎温度均未超过50度，因此，不考虑热损耗，去掉最高最低两次实测结果。3次求平均值，结果见下图表格。

实测小牛 M1 单人骑行室外环境干路面的制动距离是 1.85m 。与小牛 N1 的制动性能相比，M1 还是稍稍差了点，更轻的车身应该惯性更小。也可能是由于这辆工程试装车只行驶了 30km，前闸片还没有磨合好的缘故，行驶中前刹还有磨合初期的嚓嚓声。

国家标准化管理委员会颁布的《电动自行车通用技术条件》中规定的电动车 20km/h 制动距离应不大于 3.8m 。

小牛电动车的1.85米制动距离成绩相当不错。骑行制动距离和重量有关，所以胖子骑行，距离会大于这个结果。

车设尺寸偏窄，腿部空间宽裕，踏板脚部空间一般。比较窄的车身有其独特的优势，在堵死的车流中穿梭游刃有余。

Vitas 提醒骑友们：正确佩戴头盔等防护器具，遵章守法谨慎骑行，也是人身安全的重要保障。

▲车的安全

《小牛管家》App 2.0马上会登录，将有更多功能待体验。

M1 在座桶电池仓还为锂电池留下了锁孔，待厂家推出锁具。

小牛电动 M1 依旧联合平安财产保险公司推出牛油保，丢一辆，赔一辆。

强迫症的你不用每过10分钟就对准楼下按一按遥控器了。

Vitas 建议锁车可以购买 1.2m 左右的链条锁，直接将车锁在树上或者固定物上，就算大金杯货车，也不用怕。

〓 第三章：动力、续航 〓

小牛 M1 使用博世高性能无刷永磁电机，实际使用非常的安静，M1 使用 600W 的电机代步够用，上坡过天桥也没有任何问题（爬坡测试见文尾视频）。

如果需要更强劲的动力，或者你想讲小小牛 M1 改装玩翘头，考虑 Pro 版或至少 1000W 电机。

小牛电动设计了手提式可拆卸锂电动力源，形状是一个精美考究的金属面板盒子，重量仅为 8.3kg，相比同行的铅酸电池一般 50kg 的重量，女生都能轻易操作。小牛 M1 Pro 版续航里程因而达到惊人的 120km，小牛电动车的超长续航里程，在任何城市中往返无需充电，经理600次充放电后，正常频率使用三年依然能保有70%的电量，相比采用铅酸电池的产品，这一数据只有10%。

5年的锂电寿命和1.5年的铅酸电池寿命，小牛电动车的最大使用周期、长期使用下的可靠性有了显著提高，与松下锂电池配套的，小牛电动还拥有 BMS 电池管理系统，既充分发挥了电池的性能，也提供了过充的保护、过放保护、电池均衡、温度保护、短路保护、过流保护六重安全保障。

小牛电动踏板车配置特斯拉同级的松下18650锂电池，这种拥有超高能量密度，工业批量生产一致性好的锂电动力，为小牛电动踏板车带来了革命性变化。

都市版 M1C 使用 2.8A 充电器，没有风扇靠外壳散热，晚上充电没有任何声响。

充满电需要5小时左右，而售价6299的 Pro 版充满电只需要3小时。

48V-20Ah 的 M1c 电池满电电压为 53.2V 。

小牛电池轻松拎回家，电池，金属外壳更像个工艺品。

电池上的按键按压之后，会显示剩余电量，又一人性化设计。充电接口与 N1 的不同，想买 M1 电池放在 N1 座桶的朋友，可以考虑其它接线方案。

晚上可以放在阳台充电，再也不用从3楼放个很长的插线板下来了，其实那样很不安全。

▲M1C 都市版 80km 实测续航里程：

4月24日在北京亲测，体重 66kg，线路： 西单地铁站-宋家庄顺五条附近，地图显示 11.5km ，实际路况红绿灯上下坡等，最高速 40km/h，一直使用2档动力档，电量37%-19% ，掉电18% 。

几日体验，电池电量掉电比较线性，以此，M1 都市版满电可行驶 62km。

如果使用1档节能档，速度最高限制 20km/h，续航应该会大于70公里(待测）。

仅 8.3kg 如此小巧轻便，做的像工艺品的锂电池，续航超出预期。

〓 第四章：骑行感受 〓

经过几天的骑行体验，小牛 M1 给我的最大便利是可以自由穿行在已经堵死的车流中（如每天晚高峰的刘家窑桥），由于车身窄，小巧轻便，车辆间很窄的空隙就能通行，实在不行就上道沿通过。那些坐在车里的，只能满头汗水焦急的看着小牛消失的尾灯。

实测小牛 M1 最高可通过 15.5cm 高的台阶不会托底，通过性还不错。（详情见文章末视频评测）

超强的续航可以让自己的出行圈更远一点，再远一点。美丽四五月骑电动车出去真的很舒服。

与身体接触的各个部位都符合人体工程学设计，座椅舒适，腿部空间充裕，唯一感觉手把稍有偏粗。个人更喜欢像 N1 大小的手把尺寸。

实测 M1 工程试装车手把尺寸 34mm，而手感更好的 N1 尺寸是 30.5mm。手大骑友另说。

按照正常逻辑，在驾驶中前方有行人或者需要警示对方场景，正确的方式是保障安全先将车速降下来，即先捏刹车再按喇叭或者闪灯操作。但是 M1 的手把较粗，加之喇叭位置距离稍远，捏着手把的时候我的手是没法很舒服的按压喇叭按钮。也希望厂家能在量产车上适当做优化。

避震方面，M1 使用铝钛合金后摇臂搭配单管中置避震器，实际测试偏软，过一些细小的颠簸路段（如盲道）过滤的比较细腻，手上的手表依旧稳稳贴合在手臂上不会被颠簸的跑偏。

减速带以 20km/h 速度通过，避震器收张及时，通过后没有多余的弹跳，还算舒适。但是较软的避震会很容易触底，体重较重者，工程试装车的后避震可能会直接触底。

专门将 M1 骑行至非铺装路面，骑行感受很舒适，通过一些凹坎，感觉传递的起伏感很柔软（详情见文尾视频评测）。

轻便小巧的车身，让 M1 更快捷更好玩，如果你是个“翘头”爱好者，1200w 电机 +120km 续航应该会给你个不错的答卷。

期待 M1 量产车更优的贴心转配。

以豪华汽车般工艺努力做好一辆电动车的小牛电功，应该会成为电动车行业的骄傲。

如今街头的小牛电动越来越多，好几次碰到同骑小牛的骑友不管是否认识，都会鸣喇叭打招呼。感觉脚下的不止是一辆电动车，是年轻人时尚、浪漫、追求的写照。

个人觉得 M1 的黄色和白色更好看，有人说M1更适合女士和学生党骑，萝卜青菜吧。
