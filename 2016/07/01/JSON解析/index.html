JSON解析

1.JSON

JSON(JavaScript Object Notation),轻量级的数据交换格式,是基于JS的一个子集,是完全采用独立语言的文本格式,容易阅读和编写,非常适合于服务器和客户端的交互

2 注意：标准的JSON格式：key必须用双引号

3 在ios常见的JSON解析方式：

3.1 原生（自带）：NSJSONSerilazation（性能最好）
3.2 第三方：JSONKit

4 JSON - OC 两者对应的数据类型

（1）JSON中的大括号{}，OC：字典、
（2）JSON中的方括号[]，OC数组吧
（3）JSON中的双引号“”，OC字符串

5 JSON之系统方法

//  1 获取文件路径
    NSString *path = [[NSBundle mainBundle]pathForResource:@"JSON" ofType:@"txt"];

    //2 创建数据对象
    NSData *data = [NSData dataWithContentsOfFile:path];

    //3 解析JSON数据,使用系统方法
    /*
     [JSONObjectWithData:放入数据对象源
     options:NSJSONReadingMutableContainers 表示解析完成后返回数值类型为NSMutableArray
             NSJSONReadingMutableLeaves   表示解析完成后返回数值类型为NSMutableString
            NSJSONReadingAllowFragments  允许JSON串最外层既不是数组也不是字典，但必须是JSON有效片段
     rror:错误信息]*/
    NSArray *array = [NSJSONSerialization JSONObjectWithData:data options: NSJSONReadingAllowFragments error:nil];
    self.dataArray = [NSMutableArray new];
   //4 便利数组，放入字典
    for (NSDictionary *dic in array) {
        //5 初始化Model类
        Student *stu = [Student new];

        // 6 使用KVC赋值
        [stu setValuesForKeysWithDictionary:dic];

        //7 将赋值后的model类放入数据源数组、
        [self.dataArray addObject:stu];
        for (Student *stu in self.dataArray) {
            NSLog(@"%@",stu);
        }
    }
}
