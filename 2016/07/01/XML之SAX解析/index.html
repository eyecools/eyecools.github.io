XML之SAX解析

解析是什么?

 从事先规定的格式中提取数据
 解析前提:
 1.提前约定好格式
 2.数据提供方按照格式存取数据
 3.数据提取方按照格式获取数据

 iOS中常见的解析格式有 :XML Json

 XML:
 可扩展标记语言,主流的数据格式之一

 XML主要作用:
 1.用作配置文件
 2.用作数据交换
 3.内容管理

 XML格式结构:
 频记忆回忆
 概念:标签 节点 根节点 数据项

 xml解析方式:
    SAX DOM(安卓还有一个PULL解析)

 SAX解析 :Simple API for Xml
 基于事件驱动的一种解析xml的方式.逐行解析数据,采用代理回调方式

 处理过程:
 开始文档->开始标签->解析数据项->结束标签->结束文档

 SAX是系统推荐的方式
  NSXMLParser是系统提供的XML解析的工具类,采用的就是SAX解析方式

  ``` bash  
     //1 获取文件路径
NSString *path = [[NSBundle mainBundle]pathForResource:@"XML" ofType:@"txt"];

//2 通过路径创建一个Data对象
NSData *data = [NSData dataWithContentsOfFile:path];

//3 创建SAX解析使用的对象

NSXMLParser *xmlParser = [[NSXMLParser alloc]initWithData:data];

//4 遵循代理
xmlParser.delegate = self;

// 5 开始解析
[xmlParser parse];

```

开始解析文档

-(void)parserDidStartDocument:(NSXMLParser *)parser{
    //准备数据，自动根据数据元素开辟空间（默认为0）
    self.dataArray = [NSMutableArray arrayWithCapacity:0];
}

开始解析标签

-(void)parser:(NSXMLParser *)parser didStartElement:(NSString *)elementName namespaceURI:(NSString *)namespaceURI qualifiedName:(NSString *)qName attributes:(NSDictionary<NSString *,NSString *> *)attributeDict{

    //    获取到当前正在解析的标签，给标签属性赋值
    _currentElement = elementName;

    if ([elementName  isEqualToString:@"student"]) {
#warning 创建并初始化Model，如果标签是student，那么创建model类，并且将model添加到数组源当中（如果不是，则不创建，model的预建立）
        Student *stu = [Student new];
        [self.dataArray  addObject:stu];
    }
}

循环读取标签的值，这个方法，得到节点里储存的信息

-(void)parser:(NSXMLParser *)parser foundCharacters:(NSString *)string{
    //将值存入model类对应的属性中，这个方法循环执行，我们需要取得是最后一个【<student></student>是按照顺序扔进数组里，挨个进行取值，所以只取最后一个，说明全部取完】
    Student *stu = self.dataArray.lastObject;
    //NSLog(@"%@",self.dataArray.lastObject);

// 根据当前已经解析出来的标签跟stu的属性对应，赋值给stu中的属性
    [stu setValue:string forKey:_currentElement];
}

结束标签解析

-(void)parser:(NSXMLParser *)parser didEndElement:(NSString *)elementName namespaceURI:(NSString *)namespaceURI qualifiedName:(NSString *)qName{
    //没有标签，解析结束
    _currentElement = nil;
}

结束解析文档

-(void)parserDidEndDocument:(NSXMLParser *)parser{
    for (Student *stu  in self.dataArray) {
        NSLog(@"%@",stu);
    }
}

错误报告

-(void)parser:(NSXMLParser *)parser parseErrorOccurred:(NSError *)parseError{
    NSLog(@"错误报告：%@",parseError);
}
